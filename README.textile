h2. Installation Notes:

h3. Needed:

* Java (also in your $PATH)
* "YUI-Compressor":http://yuilibrary.com/downloads/#yuicompressor
* Ruby
* OS: currently just Linux supported

h3. Configurate:

* Set your path to YUI-Compressor jar in rb-file  "(Get it)":http://yuilibrary.com/downloads/#yuicompressor
* Set your webOS-project directory name
* Place the rb in the parent directory of the webOS-Project

h2. Compress JS and build IPK:

pre. $ ruby build.rb
